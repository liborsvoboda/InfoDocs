## Commands
|     |     |
| --- | --- |
[azdata bdc spark status](reference-azdata-bdc-spark-status.md) | Spark service status commands.

[azdata bdc spark statement](reference-azdata-bdc-spark-statement.md) | Manages Spark statements.
