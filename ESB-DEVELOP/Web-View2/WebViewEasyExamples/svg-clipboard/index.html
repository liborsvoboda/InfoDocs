Support for the SVG format in the Async Clipboard API

This page demos support for the SVG format in the Async Clipboard API. Support for SVG makes it possible for web apps to participate in SVG file copy/paste operations with other web and native apps, such as:

  • Copying SVG content from a webpage to a native app like PowerPoint.
  • Copying SVG content from a native app like the Windows Explorer to a web app.
  • Copying SVG content between two web apps.

Your browser supports SVG format in the Async Clipboard API.
